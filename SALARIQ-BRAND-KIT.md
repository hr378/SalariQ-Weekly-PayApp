# SalariQ Brand Kit

## Brand Name

`SalariQ`

## Motto

`SMART PAYROLL. HAPPY PEOPLE.`

## Core Palette

- Deep Navy: `#151A2F`
- Aqua Mint: `#72D6D0`
- Mist Grey: `#C6CFDF`
- Soft White: `#F6F6F4`
- Border Neutral: `#D9DEE8`

## App CSS Variables

```css
:root {
  --brand-navy: #151A2F;
  --brand-mint: #72D6D0;
  --brand-mist: #C6CFDF;
  --brand-white: #F6F6F4;
  --brand-border: #D9DEE8;

  --bg: #F6F6F4;
  --panel: #FFFFFF;

  --text: #151A2F;
  --muted: #6F7A8E;

  --accent: #72D6D0;
  --accent-deep: #151A2F;
  --accent-soft: #EAFBFA;
}
```

## Color Usage

- `#151A2F`: headings, sidebar, body text, logo base
- `#72D6D0`: primary accent, CTA buttons, `Q` highlight, tagline
- `#C6CFDF`: secondary highlight, subtle logo layer, muted UI accents
- `#F6F6F4`: background and soft neutral surfaces
- `#D9DEE8`: borders, dividers, input outlines

## Typography Direction

- Brand wordmark: serif or elegant transitional heading style
- UI text: clean sans serif
- Motto: uppercase with wide letter spacing

## Logo Structure

- Stylized layered `S` mark
- Top curve in aqua mint
- Middle curve in mist grey
- Bottom curve in deep navy
- Wordmark with `Salari` in navy and `Q` in aqua mint

## Product Voice

- Clear
- Calm
- Efficient
- Human
- Modern

## Suggested UI Feel

- Minimal white space
- Soft panels
- Dark navigation rail
- Clean tables with generous spacing
- Aqua highlights instead of harsh corporate blue
